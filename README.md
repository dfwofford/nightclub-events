# Nightclub Events

WordPress plugin for nightclub and venue event management.

## Installation (Easy!)

1. Click the green **Code** button above
2. Click **Download ZIP**
3. Go to your WordPress admin: **Plugins → Add New → Upload Plugin**
4. Choose the zip file you downloaded
5. Click **Install Now**
6. Click **Activate**

Done! Go to **Events** in your WordPress menu to start adding events.

## Features

- **Event Management** - Create single or recurring events
- **Dual Flyer Support** - Upload two flyers per event  
- **Tickets & Guest List** - Add ticket links and guest list URLs
- **Calendar View** - Monthly calendar showing all events
- **Display Settings** - Customize colors, fonts, and themes
- **Elementor Widgets** - Drag-and-drop event widgets

## After Installing

1. Go to **Events → Add New** to create your first event
2. Go to **Events → Display & Themes** to customize the look
3. Add the calendar page using the **[nc_calendar]** shortcode
4. Add event flyers to any page using **[nc_flyers]** shortcode
