== Description ==

The ultimate WordPress plugin for nightclubs, bars, venues, and event promoters. Manage events, flyers, tickets, guest lists, and more!

**Features:**

* Create single or recurring events
* Upload dual flyers per event
* Ticket and guest list integration
* Beautiful calendar view
* 10+ built-in themes
* Elementor widgets
* Customizable display settings

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/nightclub-events/`
2. Activate the plugin through the Plugins menu
3. Go to Events → Add New to create your first event

== Frequently Asked Questions ==

= How do I add a calendar to my site? =

Create a page and add the shortcode [nc_calendar]

= Can I customize the colors? =

Yes! Go to Events → Display & Themes to customize everything.
